# 🛠 CODE-GEN (Experimental Prototype)

> **Disclaimer:** Проект собран «на коленке» за вечер для автоматизации рутины. Код сырой, баги присутствуют, архитектура — чистый хаос. Использовать на свой страх и риск.

C0DE-GEN - это экспериментальный автономный агент для терминала. Я сделал его, чтобы проверить, может ли LLM сама фиксить свои ошибки в консоли в реальном времени. 

## ⚠️ Текущий статус:
- **WIP (Work In Progress):** Проект не доделан. 
- **Баги:** Может зациклиться, может не понять путь к файлу, может попытаться удалить системную папку (шучу, но будьте осторожны).
- **Код:** Написан быстро, оптимизация отсутствует как класс.

## 💡 Зачем это нужно?
Несмотря на «костыли», утилита умеет:
- Входить в итеративный цикл (до 40 попыток) для решения задачи.
- Читать логи ошибок прямо из терминала и предлагать фиксы.
- Работать с файловой системой через Gemini 2.0 / DeepSeek.
